GGally是一个功能强大的可视化包，是可以集相关关系图、箱线图、直方图等于一身的R绘图包，现在复现这篇GGally绘制的几个图形 - 知乎 (zhihu.com)的图

GGally绘制简单平行坐标图

使用iris数据进行绘制，函数是ggparcoord()

library(GGally)
## 载入需要的程辑包：ggplot2
## Registered S3 method overwritten by 'GGally':
##   method from   
##   +.gg   ggplot2
ggparcoord(data = iris,
           columns = 1:4,
           groupColumn = "Species",
           showPoints = TRUE) +  ##添加点
  scale_color_brewer(palette = "Set2")

通过设置“splineFactor = TRUE”参数，绘制平滑的平行坐标轴；

ggparcoord(data = iris,
           columns = 1:4,
           groupColumn = "Species",
           splineFactor = TRUE) +
  scale_color_brewer(palette = "Set2")

添加箱线图

ggparcoord(data = iris,
           columns = 1:4,
           alphaLines = 0.2,
           showPoints = TRUE,
           boxplot = TRUE)

更改x轴变量的排序

ggparcoord(data = iris,
           columns = 1:4,
           groupColumn = "Species",
           order = "anyClass") + ##还可设置成allClass、skewness、Outlying
  scale_color_brewer(palette = "Set2")

可以使用facet wrap函数为每个组创建一个平行坐标图

ggparcoord(data = iris,
           columns = 1:4,
           alphaLines = 0.2,
           boxplot = TRUE,
           groupColumn = "Species",
           order = "anyClass") +
  scale_color_brewer(palette = "Set2") +
  facet_wrap(~ Species)

GGally绘制矩阵散点图

使用ggscatmat()函数绘制散点图

ggscatmat(data = iris, columns = 1:4, color = "Species", alpha = 0.8)+
  theme_bw(base_size = 14)+
  theme(plot.title = element_text(hjust = 0.5))
